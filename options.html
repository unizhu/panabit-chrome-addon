API & IP / Domains Settings

Configure your API settings, including your API key and allowed domains.

Prerequisites

This Extention requires Cross Domain Request , However , Chrome does not allow Cross Domain Request by default.

So , You need to install a Chrome Extension called Allow CORS: Access-Control-Allow-Origin to enable Cross Domain Request.

Or you can install similar extension from Chrome Store.

Intall Extension Search Chrome Store
Recommendation

API Key is optional , But still recommed to register an API key from https://my.maclookup.app/

IP & Domains

E.g. beach_access192.168.0.200 for IPv4 , beach_access2001:0000:130F:0000:0000:09C0:876A:130B for IPv6 , beach_accessmy.server.example.com for Domain
